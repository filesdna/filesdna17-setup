Odoo SaaS Rocket Kit from Pragmatic can be deployed to the cloud in an auto-scaling mode. You can scale the architecture to suit your requirements. If you want your clients to have uninterrupted access to your servers, you can choose to have multiple servers.

Frequently Asked Questions

Qn : I have a base Odoo database with the modules, certain configurations per module, etc. Is it possible for the module to create a copy of this base Odoo database when the customer subscribes to the service in order to create a customer instance?
Ans : If you are going with the container/ docker approach, then this is not possible. If you are going without a container, then we use one bare database to replicate the bare minimum modules to be installed in every new tenant database.
Qn : After a customer purchases a plan, are the credentials sent to the customer via email? or is there an option to send SMS instead of mail? If yes, do I need to modify the system for that?
Ans: We send notifications via email. If SMS is to be sent, then we can do that with extra development hours.
Qn : Is it possible to send reminders and notifications such as the expiration date of the plan to customers via SMS? If yes, what do we need?
Ans : We send notifications via email. If SMS is to be sent, then we can do that with extra development hours.
Qn : Instead of using Odoo's website, can we use Woo Commerce?
Ans : Our module supports only Odoo website.
Qn : How can I know from the Tenant Database List which customer subscribed to what/which plan?
Ans : We maintain a separate record of an agreement which keeps the records of products purchased by the customer for each of the tenant database.
Qn : Do you provide a backup solution for individual instances?
Ans : Our SaasKit provides database backups.
Qn : Can I invoice the backup based on usage?
Ans : No, you cannot invoice the backup depending on usage.
Qn : What does Installations (with third-party applications) With Reasonable Charges mean? Does this imply I can't resell third-party apps?
Ans : You cannot resell third-party apps. You may, however, provide it as a service on the tenant Database.
Qn : Do you have a full guide documentation about installation and configuration of system requirements and the module configuration also ?
Ans : We do not provide installation guide. We provide only video that shows how to use system.
Qn : In the SaaS Kit Deployment Options Table , i see the multiple options with differnt cost , i need to understand. are these options consider a additional modules to the base module of SaaS Kit OR these are installation and configuration Fees ?
Ans : There is an installation fees based on different architecture customer go with.
Qn : Can I clone an existing database to create a new tenant database?
Ans : Yes, this is possible ONLY from the backend forms.
