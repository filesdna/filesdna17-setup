Frequently Asked Questions

Is your saas system works with multi server and different versions of odoo?

Yes. you can connect with unlimited server system and can launch tenants in different versions of odoo

How can import module from frontend?​

You can import module by accessing the menu named import module for all tenant and saasmaster

How can configure own domain in particular tenant?

You can configure domain from my instance option from portal user interface

What is the prerequest of domain masking? 

 Need to purchase a domain and SSL certificate for https protocol

How to import bulk modules in tenants? ​

You can import module by accessing the menu named import module for all tenant and saasmaster, if it is not works, then you can connect with terminal and update it

How can delete a complete environment?

All tenant can be delete complete environment by purging the database.

How can update the users? 

You can update user from my instance option from portal user interface.

How can add addons module in existing tenant?

You can add addons module by accessing the import module menu from existing tenant and saasmaster.

If the tenant failed due to connection time out what to do?

From saasmaster,we can change the time for creation and cloning times for tenants in saas settings.

How can download the database? 

You can download it from my instance option from portal user interface.

How can add multiple server?

You can add server in server list in saas master.

How can restore the database?

You can restore it from my instance option from portal user interface.

How can restart the service? 

You can have a menu named container action for restarting container from both tenant and saasmaster.

How can test connection when a tenant is down? 

Connect tenant server from ssh ,then restart the tenant from the server location, check the log with docker-compose up command.

If you get already allocated port? what to do? 

Delete unwanted port(port which is not allocaed by a tenant name) from custom ports menu in saasmaster.

How can install the libraries and modules?

Connect the server , connect the container ,docker exec -it --user root container_name /bin/bash then we can install the libraries - apt-get upgrade.

How can connect the tenant from user? 

By accessing tenant server, docker exec -it --user odoo container_id /bin/bash

How can check the logs of tenant and master? 

Connect the tenant server from ssh,execute docker logs --follow tenant_name

How to find backup location of of tenant and master ?

We can check it from configure backup menu in saasmaster and tenant.

How can export invoice? 

We can access an export all from the action enu in invoice module.

How can replicate tenants? 

In saasmaster, from cloning wizard we can replicate the tenant.

How can rename domain? 

In saasmaster, we can access change url from tenant database list menu, or we can change by accessing add domain from customer portal.

How can update the config?

Access the tenant config file from destination path of tenant server,and update the config, after that restart the container.

How can update the module in base? 

Access the tenant environment, update the command in compose file with /usr/bin/odoo -d database_name -u module_name,restart the container.

How can update the module for each of customer requirements? 

By accessing the module upgraede option from saasmaster and select the tenant needed and load the modules, then select the module which you want to upgrade.
