Face Id Authentication | Login Face Recognition | Login Face Authentication

This module allows the odoo users to login using Face Recognition. This functionality works in conjunction with User Employees Photograph while Login.

1. Face Id Authentication



Features

- Anti-spoofing from Photos
- Login utilizing Face Recognition
- Face Recognition On Login
- Faster detection after load the models



Installation

- Copy faceid_authentication module to addons folder

- Install the module normally like other modules

1. Face ID Configuration

Settings --> Users --> Face Recognition - Add



2. Capture / Upload User Photographs on Users Master

Settings --> Users --> Face Recognition - Add




Capture Snapshot and wait for Load the Face Descriptors. (It will take some time to capture the Photo)





3. Login Using Face Recognition

Login -> Face ID Login



Popup will open for Recognise Employee Face


After successfull Authentication, the page will redirec to Home page
