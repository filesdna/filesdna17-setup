With readonly permission you can't able to Add/Update/Delete dashboards.

To get full permissions, you have to grant permission in category AnalytiX Dashboard Access with group Show Full Dashboard Features

What is AnalytiX Dashboard?

AnalytiX Dashboard is a cutting-edge Odoo module designed to transform how you visualize and analyze data. It offers intuitive, interactive, and insightful dashboards that empower users to navigate their business landscapes confidently.

Can base module views be integrated into the dashboard?

Absolutely! Base module views like forms, trees, kanbans, calendars, and pivots can be embedded into the dashboard with applied filters, allowing for a comprehensive view of relevant data.

Does AnalytiX Dashboard offer advanced date filters?

Yes, it provides advanced date filters to load dashboard data with predefined filter types, enabling users to access data based on specific date ranges or criteria.

How does the dashboard facilitate data retrieval through queries?

Users can seamlessly access and analyze specific data sets by crafting custom queries tailored to their business needs. This feature allows for detailed insights, information filtering, and targeted analysis.

Is it possible to share dashboard statistics externally?

Yes, users can share business statistics with customers and business groups by downloading dashboard items and the full dashboard in PDF format or by embedding the dashboard link on other websites.

How does AnalytiX Dashboard manage user permissions?

Access controls are simplified by applying user groups, allowing for read-only or full access to dashboards. This feature ensures secure and personalized data visibility for different users or teams.

Can dashboard items be rearranged or duplicated between dashboards?

Yes, users have the flexibility to effortlessly rearrange and duplicate widgets between dashboards, enabling dynamic data presentation and management.

Does AnalytiX Dashboard offer customization options for data visualization?

Absolutely! It provides various design options such as 9 tile item styles, 3 trend chart designs, 5 KPI progress designs, 4 list view designs, and 10+ theme color palettes, allowing users to create visually appealing dashboards aligned with their brand aesthetics.
