Cloud Storage Replacing Local Storage

Local Storage becomes complex; especially when you need to store data in bulk. On the other hand, Cloud storage is more scalable; hence, you can increase or decrease space as needed easily. Moreover, data stored on the cloud is easily accessed on any device that has an internet connection.

What Draws Attention Towards Cloud Storage?

Data Accessibility:

  • If your data is stored in Cloud then no matter where you are, you can access your data.

Scalability:

  • Cloud Storage offers high scalability as per the requirement of your business; so you can scale up and down as per your requirements.

Cost Effectiveness:

  • Cloud Storage eliminates the cost of handling local servers. The cloud storage works on the model Pay-for-what-you-use which allows you to scale up and down as per your requirements.

Data backup and Recovery:

  • Managing data on your local server is prone to risk as you may face complete data loss due to the malfunctioning of the hardware. Whereas in cloud storage, discrete amounts of data are copied from one place to another to recover when production is down.

Google Cloud Storage for Odoo Data

Google’s cloud platform offers reliable and highly scalable cloud computing services to its users. These services help clients compute and store data; and help developers build, test, and deploy apps. Google Cloud Storage is trendy as it fits most of all criteria to match the perfection. Your business requires Google Cloud Storage for smooth functioning; no matter where you are. Now, Odoo users can store all their massive Odoo Data in Google Cloud Storage. The module Odoo Google Cloud Storage syncs your Odoo data to google cloud storage. So, you can access your data anywhere and anytime; No more dependencies on Physical Locations!

Features

Sync Your Odoo Data to Google Cloud Storage

  • Upload the Odoo attachments to Google Cloud Storage.
  • Run a script file with google credentials to sync or restore the data.

Access Your Odoo Data from Anywhere

  • Physical Location is no more a concern if data in synced to Google Cloud Storage.

Automatically Deletes Unwanted Data

  • No need to worry about the Unwanted Data Records to be stored in Google Cloud.
  • The attachments are automatically Deleted from Google Cloud when they are deleted from Odoo.

Set Access Rights As You Want

  • The admin can set the access rights for the Google Cloud bucket/object.
  • The module allows setting expiration intervals to the attachments under ‘Not Public’ access.

Use the Storage for Multiple Odoo Databases

  • It can be used as external storage for any number of databases.

Decreases the Load in Odoo

  • As your Odoo data can be stored in Cloud then it automatically decreases the Load at Odoo end.

Google Cloud Storage Configurations

  • Select the google cloud storage json key file.
  • Add google cloud storage bucket name and activate transfer.
